Sistema de Gestión de Ventas para Farmacia (Desarrollado en JavaFX)

**Descripción del Proyecto

Este proyecto tiene como objetivo la creación de un completo sistema de gestión de ventas específicamente
diseñado para una farmacia.

Mi responsabilidad principal en este proyecto será desarrollar el módulo de software encargado del manejo del 
inventario de productos.

**Módulo de Manejo de Inventario de Productos

El módulo de manejo de inventario es un componente crítico del sistema y tiene como propósito principal 
proporcionar a los usuarios la capacidad de gestionar eficazmente el catálogo de productos de la farmacia.
Las principales funcionalidades que abordaremos en este módulo son las siguientes:

1. Agregar Nuevos Productos:
Los usuarios podrán introducir y registrar nuevos productos en el sistema.
Para cada producto, se podrán especificar detalles esenciales como el nombre, precio, cantidad en stock,
categoría y una descripción detallada.

2. Modificar Productos Existentes:
Se permitirá la actualización de información para los productos que ya se encuentren en el inventario.
Esto incluye la capacidad de modificar detalles como el nombre, precio, cantidad en stock, categoría y descripción,
asegurando que los registros estén siempre actualizados y precisos.

3. Eliminar Productos:
Cuando sea necesario, se podrá eliminar productos del inventario. Esta función es fundamental para gestionar de manera efectiva los productos que
ya no se ofrecen o que han sido descontinuados.


** Importancia de la Gestión de Inventario

El control adecuado del inventario es esencial para el funcionamiento eficiente de una farmacia. A través de este sistema desarrollado en JavaFX,
aseguramos que los productos estén siempre disponibles para su venta y que se mantenga un registro preciso de todos los elementos en stock.
Esto no solo mejora la eficiencia operativa, sino que también garantiza la satisfacción de los clientes al proporcionarles información actualizada y
productos disponibles.

**Tecnologías Utilizadas

Este sistema se desarrollará utilizando JavaFX, una tecnología robusta y ampliamente utilizada para la creación de interfaces de usuario en aplicaciones de escritorio. 
JavaFX proporciona una plataforma versátil para desarrollar interfaces de usuario interactivas y modernas, lo que asegura una experiencia de usuario atractiva y
eficiente.
